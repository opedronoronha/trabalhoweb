O que é CSS?

CSS (Cascading Style Sheets) é uma linguagem de estilo utilizada para definir a apresentação de documentos HTML. Ele permite que você aplique estilos como cores, fontes, espaçamentos e layouts às suas páginas web.

Aplicações do CSS

Propriedade Descrição
Cor de fundo Altera a cor de fundo dos elementos.
Fonte Define o estilo, tamanho e cor da fonte utilizada no texto.
Layout Permite organizar os elementos em diferentes posições na página.
Espaçamento Controla margens e preenchimentos ao redor de elementos.

Como usar CSS

Existem três maneiras de usar CSS:

  • Inline: Aplicar diretamente nos elementos HTML com o atributo style.
  • Interno: Definir no cabeçalho do documento HTML usando a tag <style>.
  • Externo: Criar um arquivo CSS separado e vincular ao HTML com a tag <link>.
