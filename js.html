O que é JavaScript?

JavaScript é uma linguagem de programação que é amplamente utilizada para criar interatividade em páginas da web. Enquanto HTML define a estrutura e o CSS cuida da aparência, o JavaScript permite que você adicione comportamento dinâmico ao seu site, como responder a eventos do usuário, validar formulários, atualizar o conteúdo da página sem recarregá-la e muito mais.

Aplicações do JavaScript

Propriedade Descrição
Interatividade Ele pode reagir a eventos do usuário, como cliques de botão, movimentação do mouse e digitação em campos de texto.
Manipulação do DOM JavaScript permite modificar a estrutura HTML e CSS de uma página em tempo real, o que possibilita, por exemplo, criar animações, alterar estilos ou adicionar/remover elementos da página.
Validação de formulários Antes de enviar dados para um servidor, o JavaScript pode verificar se os campos do formulário foram preenchidos corretamente.
Comunicação com servidores Através do JavaScript, é possível enviar e receber dados de um servidor sem precisar recarregar a página (usando tecnologias como AJAX).

Como usar JavaScript

Primeiros passos para usar JavaScript:

JavaScript Inline: Salve o arquivo com a extensão .html (ex: index.html)

JavaScript Interno: Adiciona o código JavaScript diretamente dentro da página HTML, entre as tags script. Esse código pode ser colocado no head ou no final do body.

JavaScript Externo: O código JavaScript pode ser armazenado em um arquivo separado com a extensão .js. Esse arquivo é vinculado ao HTML usando a tag script.
